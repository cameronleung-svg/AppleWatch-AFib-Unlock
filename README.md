1. 拔掉 iPhone 中的国内实体 SIM 卡（如果使用的是 eSIM，请在系统设置中临时**禁用该蜂窝网络**）。
2. 关闭手机的 Wi-Fi 和蓝牙。

### 第二步：使用电脑修改 iPhone 底层定位
1. 在电脑上安装并打开 **AnyGo**。

3. 在 AnyGo 的地图搜索栏中，输入支持房颤和心电图功能的境外地区（推荐：**中国香港** 或 **新加坡**）。

### 第三步：修改手机系统区域
1. 进到 iPhone 的 `设置` -> `通用` -> `语言与地区`。
2. 将 **地区（Region）** 修改为刚才你定位的对应地区（如：香港）。

### 第四步：激活房颤与健康清单

2. 打开 iPhone 上的 **健康（Health）** App。
3. 点击右上角头像进入“健康清单”，找到 **房颤提示（AFib History）**。
4. 按照屏幕提示点击设置。此时系统底层 GPS 与区域配置完全合规，即可完美激活写入账户！
